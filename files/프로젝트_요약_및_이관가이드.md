## 시스템 구성

### 기술 스택
- **Frontend**: HTML5, TailwindCSS, Vanilla JavaScript
- **Backend/DB**: Supabase (PostgreSQL)
- **API**: Google Drive API v3
- **호스팅**: 로컬 HTML 파일 (추후 Vercel/Netlify 배포 가능)

### 메뉴 구조
```

├── 메인

│   └── 프로세스현황
├── 관리
│   ├── 운영관리
│   │   ├── CS관리
│   │   ├── 바이어관리
│   │   ├── 상품/리스팅관리
│   │   └── 견적/협상관리
│   └── 확장/거래관리
│       ├── 주문관리
│       ├── 배송관리
│       ├── 이슈관리
│       └── 정산관리
├── 성과
│   └── 성과관리 (KPI 대시보드)
└── 설정
    ├── 자료집 (MOQ, 가격, 리드타임, 인증서, 시험성적서)
    └── 응답 템플릿 (바이어 응대용)
```

### 1. CS관리

- 상품정보, MOQ/가격, 리드타임, 인증서, 거래조건 탭

- 클립보드 복사 기능

### 5. 응답 템플릿
- 첫인사, 문의응대, 견적안내, 샘플안내, Follow-up, 클레임대응

```json
{
  "mcpServers": {
    "playwright": { ... },
    "terminal": { ... },
    "googleSearch": { ... },
    "n8n-mcp": { ... },
    "supabase": {
      "command": "npx",
      "args": ["-y", "@supabase/mcp-server-supabase@latest", "--access-token", "sbp_4f325b6cfe4f125fd306ea730f9ca1bc6a3faf89"]
    }
  }
}
```
